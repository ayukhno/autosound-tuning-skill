# Метод Хашімото — by-ear побудова сцени (альтернативний/доповнюючий підхід)

Цілісна **by-ear** методика японського інсталятора **Хашімото** (Hashimoto; переклад Антона Ніколаєва, відео-конспект). Ціль = **побудова звукової сцени**. Це АЛЬТЕРНАТИВА/доповнення до нашого REW-вимірювального процесу (`process-phases.md`) — корисна там, де працюємо на слух (Фаза 5) або без повного REW. Сигнатурний прийом: **«спершу оптимальна КРУТИЗНА фільтра, потім оптимальна ЧАСТОТА»**, кожен стик окремо, кожна сторона окремо.

## Порядок: кросовери(slope→freq) → полярність по смугах → затримки(вертикалі→центр) → EQ(останній) → войсинг на музиці
0. **Прогрів компонентів** (нові — обкатати як двигун; інакше при зведенні мідбас/середина/саб «розвалюються» по черзі). Грубий старт-тюн → обкатка → точний.
1. **Первинні значення** (щоб «заграло»): процесор default; маршрутизація каналів. Стартові кроси: **саб↔мідбас 60-80 · мідбас↔СЧ 200-1000** (велика дифузорна СЧ — нижче, мала/купольна — до 1000) · **СЧ↔ВЧ 3-6к** (2-смуга: мідбас↔ВЧ 3.5-6к).
2. **Первинні затримки:** виміряти від центру кожного драйвера **до кінчика НОСА** (голова на підголівнику → ця точка між вухами). Ref = найдальший драйвер (саб); решта = дельта.

## Полярність — по СЛУХУ, по смугах ⚠️ ВАРІАНТ
Моно-вокал. Лишити тільки **мідбаси** → образ має сфокусуватись ~центр. Якщо розвалюється/липне до динаміків — **перевернути полярність ОБОХ** (іноді лише ОДНОГО; або перевірити підключення). Потім те саме для **СЧ**, **ВЧ — останніми** (там λ ≈ відстань між вухами → фазу важко визначити, повернутись пізніше). Образ зміщений → трохи затримками (вліво → відсунути правий).
> ⚠️ **СУПЕРЕЧИТЬ Wehmeyer/нашому §16** («полярність НЕ ставити по центр-образу пар — приведе до 180° протифази після затримок»). Тримаємо як **варіант** (Хашімото робить це РАНО для грубого старту, потім уточнює фазу ВЧ по злиттю після вибору крутизни). Перевірити сумацією (`diagnostic §9`); якщо конфлікт — довіряти сумації.

## Фільтри: КРУТИЗНА → ЧАСТОТА, кожна сторона окремо
**Чому L/R окремо і параметри можуть РІЗНИТИСЬ:** фільтри крутять фазу, драйвери під різними кутами → чуєш по-різному. **Велика хиба — робити крос-частоти/порядки однаковими L/R.** Орієнтир — СЛУХ, не цифри. *(Узгоджується з нашим асиметрія-кросовера §15.)*
- **САБ (першим):** LPF off, повна смуга. Субсонік HPF ~20 Гц — міняти ПОРЯДОК, слухати контрабас 80-100 (найчіткіше/швидко/енергійно). Тоді LPF 80-100 — міняти крутизну (бас соковитий/глибокий). Тоді ЧАСТОТИ (та сама логіка). Реко: субсонік 20 Гц (вище — для ФІ); LPF не надто низько, крутизна **12 або 24 dB/окт** (6дБ → перекриття/гул; 48дБ → фазові спотворення).
- **МІДБАС (2 треки: мід/верх + мід/низ):** HPF — низ 25-32, трек-1 (мід/верх), крутизна → вокал багатий, чути нюанси+реверберацію. LPF ~10-12к, трек-2 (мід/низ), крутизна → бас чіткий/пружний. Тоді частоти: HPF слухати верх/середину, LPF — бас. HPF **на резонансі драйвера або вище** (драйвер грає краще далі від резонансу; не нижче 40-50). LPF — попередній (фінал при стику з ВЧ). Тоді **рівні саб↔мідбас** (гладкий перехід): крутити LPF-частоту саба + РІВЕНЬ саба; мідбас-LPF не чіпати (мідбас = основа).
- **3-смуга — стик СЧ↔мідбас:** частота розділу **≥ октава над резонансом СЧ**, але не вище за межу поршневий→модальний режим мідбаса. Межу знайти: трек-2, LPF 2-й порядок, знижувати частоту поки звук макс енергійний/щільний/артикульований. (Тут: частота → крутизна → знов частота.)
- **СЧ** (трек із небагатьма інструментами, повна смуга — Jennifer Warnes): HPF = LPF мідбаса, крутизна (слухати низ + верхні обертони); LPF крутизна (барабани — пружні, з атакою); на стику мідбас↔СЧ — **вокал природний/чистий**. LPF-частота попередня. Рівні СЧ↔мідбас → звучать **злито, як один драйвер**.
- **ВЧ** (трек із металевими/тарілками; мідбас+СЧ увімкнені, саб off): HPF = LPF СЧ/мідбаса, але **обов'язково вище резонансу ВЧ** (інакше грубо/брудно); вище HPF = легше/чистіше (особл. 6-12 дБ). Крутизна → тарілки живі/природні, довгі плавні послезвуки, не металево/скляно (обертони впливають на основні тони!).
  - **Полярність ВЧ — ТЕПЕР** (після вибору крутизни!): пряма vs обернена — ВЧ має **ЗЛИТИСЬ** з мідбасом/СЧ, не звучати сам по собі. Де злився = правильна фаза.
  - ВЧ-частота: лишити тільки ВЧ — удари тарілок довгі послезвуки, повітря, без зайвої дзвінкості/піску. Тоді + мідбас/СЧ: як ВЧ вливається → виставити **РІВЕНЬ** де вокал найприродніший. Корекція частот — краще крутити **фільтр МІДБАСА, не ВЧ**.
  - ⚠️ Зміна рівня однієї смуги зачіпає стик і вгору, і вниз по ланцюгу (каскад).

## Затримки фронту (Time Correction) — серце сцени
Ціль = сфокусовані локалізовані образи (як задумав звукорежисер). Моно-запис. **«Вертикалі»** = звести мідбас↔СЧ↔ВЧ кожної сторони ОКРЕМО. Саб off, пасажирська: (2-сму) стик ВЧ↔мідбас — запам'ятати вихідну затримку ВЧ, додавати малі ділення, стежити за образом → локалізується біля ВЧ як ОДНЕ джерело (зазвичай мала правка; якщо великі ділення нічого не міняють — помилки інсталяції/орієнтації/оформлення). (3-сму у 2 етапи: мідбас↔СЧ, тоді →ВЧ.) Образ на торпедо → добре, тоді водійська (складніше: панель+кермо). **Повний фронт:** моно точно по центру з однієї точки; якщо смуга розпливається — підсунути затримки драйверів (водійський бік) 1-2 ділення. **Саб:** контрабас-трек, рухати затримку саба → найчіткіше/динамічно/з напором без гулу.

## EQ — ОСТАННІЙ етап
Після кросоверів+затримок (EQ теж крутить фазу). Моно синуси/вузькі шуми різних частот → мають іти з ЦЕНТРУ. Грати тихо, низ→верх. На НЧ локалізації нема (λ > салон); ближче до верху образ збирається в центр. Образ змістився вправо на частоті → там правий гучніший → **трохи прибрати справа + додати зліва** → центр (вліво → навпаки). Пройти весь спектр. ⚠️ **На 2.5-5 кГц звук здається гучнішим/кричить — EQ там НЕ чіпати** (чутливість слуху); лише визначити відносну L/R гучність. Серйозний зсув на частоті → повернутись до попередніх етапів. Надлишок/провал на частоті = резонанси/інсталяція → в ідеалі лікувати причину.

## Войсинг на реальній музиці (емоція/характер) — = наша Фаза 6
«Академічну» систему тоді налаштувати на **улюбленій музиці** (краще живі інструменти — багатші; мозок порівнює з реальним інструментом). Характер інструмента = основні тони + **обертони/гармоніки** → впливати і на основні, і на гармоніки. Загалом яскравіше: ↓~200 + ↑~2к, тоді збалансувати гармоніки. ↑100/300/1к + гармоніки = більше енергетики. → деталі by-ear у `voicing-by-ear.md`.
> **Робити перерви** — слух забивається (нетренований швидше), перестаєш чути вплив змін.

## Що взяти в наш процес
- Прийом **«крутизна→частота, кожна сторона окремо»** — потужний для by-ear вибору фільтрів (Фаза 5 / коли REW недоступний). · Стик-частоти/порядки L/R можуть різнитись = **by-ear ВАРІАНТ** (дефолт — симетрія L/R: однакові тип+порядок+частота для стабільного фантомного центру; асиметрія лише коли симетрія не дає образ → imaging верифікувати; `filter-types §Симетрія L/R`, `diagnostic §15`). · EQ-центрування моно-синусами + **не чіпати 2.5-5к**. · Прогрів компонентів + перерви. · Полярність по смугах = **варіант** (звіряти сумацією).
