# Підготовка до змагань (EMMA / AYA / CARMusic) — критерії, треки, діагностика

Контекст коли користувач готується до суддівства (а не просто «насолода»). Формати: **EMMA**, **AYA**, **CARMusic** — споріднені SQ-протоколи. Категорії: **SQ** (sound quality), **SQL** (SQ+гучність), **SPL** (гучність). У car-контексті будь-яка з цих абревіатур + назва формату = сильний тригер.

## Imaging-діагностика через тест-трек (інструмент→частота)
Суддівські треки програють відомі інструменти на відомих позиціях сцени. Якщо інструмент **їде вбік** від своєї позиції → та частотна смуга **гарячіша на тому боці**. Це швидка карта L/R-дисбалансу по смугах (доповнює соло-замір `m-L − m-R`).

### EMMA 2024 — тест-трек позицій (рік важливий, формат міняється між сезонами)
**5 позицій:** Left · Right · Center · Left-Center (LC) · Right-Center (RC). C/LC/RC точніше скорити, ніж L/R.
**Порядок інструментів на кожній позиції:**
| # | Інструмент | Частоти | Канал DSP |
|---|---|---|---|
| 1 | Electronic Bass | 40-100 Hz | sub + midbass |
| 2 | Electronic Guitar | 100-250 Hz | midbass |
| 3 | Flute | 250 Hz–1 kHz | midbass(250-300) + mid(300-1k) |
| 4 | Celesta | 1-2 kHz | mid |
| 5 | Triangle | 2-4 kHz (+гармоніки >20k) | mid + tweeter |
**Читання:** інструмент на LC їде ЛІВІШЕ → та смуга L-hot; на RC їде ПРАВІШЕ → R-hot. (Напр.: целеста LC вліво = m-L гарячий 1-2к; трикутник RC вправо = правий ТВІТЕРНИЙ СКАТ 2.5-3.5к гарячий — перевіряти скат нижче кросовера, не лише passband.)

## AYA «Authentic Audio Check» — категорійна система (рамка діагностики)
AYA (Are You Authentic, ayasound.org + Stockfisch-Records) оцінює систему **по КАТЕГОРІЯХ — кожен трек ізолює один вимір.** Офіційний SACD (19 треків) — аудіофільні Stockfisch-записи; **категорійна схема (нім. терміни) — це і є переносна діагностична рамка**, незалежно від конкретного треку:
| Трек(и) | Категорія | Що оцінює (→ tuning-вимір) |
|---|---|---|
| 01 | L/R Channel | базова L/R перевірка/маршрутизація |
| 02 | Level Setting | референсний рівень |
| 03-04 | **Tiefbass** | глибокий бас (розширення/контроль <40-50) → саб |
| 05 · 06 | **Bass** (06=Bass/Stage Height) | бас + висота сцени → мідбас/саб |
| 07-08 | **Grundton** | фундамент/тіло (нижня-середина) → мідбас↔СЧ |
| 09-10 | **Mittelton** | середина → СЧ |
| 11-12 | **Hochton** | верх → ВЧ |
| 13-14 | Balance | тональний/L-R баланс |
| 15 | **Raum** (Width/Depth) | ширина+ГЛИБИНА сцени → staging-depth |
| 16 | Focus | фокус образу |
| 17 | Height | висота сцени |
| 18-19 | Fine Dynamics | мікродинаміка |
> ⚠️ **Регіональні AYA-style диски переймають НАЗВИ категорій** (TIEFBASS, RAUM…) **з ІНШИМИ треками.** Користувацький диск (`03-TIEFBASS`=Young Jeezy «Put On», `22-RAUM`=Mozart) — НЕ офіційний Stockfisch (там 19 треків, інші виконавці). Переносне = **категорія→вимір**, не конкретний трек. Конкретні треки користувача — питати/логувати, не припускати з номера.

## TIEFBASS / deep-bass критерій — equal-loudness і SQ↔SPL fork
- Deep-bass треки мають фундамент **нижче тестових тонів**: напр. «Put On» — контент **25-35 Гц** (найглибша нота ~16), тоді як рівневий чек грає 40-55 Гц.
- **Equal-loudness:** щоб тон звучав однаково гучно, нижчий потребує БІЛЬШЕ SPL (40 Гц ≈ +6 dB над 55 на помірному рівні; на гучному — менше). Плоский SPL → низ звучить тихіше. **Перевіряти реальну частоту треку (web) ПЕРШ ніж EQ-ити «до рівного».**
- **Судді оцінюють відтворення/авторитет глибоких нот, не буквальну рівну гучність** (на 25 Гц вухо майже глухе — це фізика, всі системи там тихіші).
- **SQ↔SPL войсинг-fork:** для чистого SQ субсонік тамлять (менше бубоніння); для deep-bass категорії — лишають/нарощують розширення. Це різні **пресети**, не один компроміс. Дзвонити рівень/розширення на самому треку (трек = еталон), на judged-рівні гучності.

## Войсинг під суддівство
- Фундамент (Фази 0-1: кросовери/TA/полярність/L-R) **curve-agnostic** — робиться раз, тримається між цілями.
- Competition-войсинг = **окремий пресет** (virtual-шар), форкнутий від зафіксованої бази. Тримати ОБИДВА (SQ-точний + competition/depth) → A/B на місці свіжими вухами, вибрати під категорію/день.
- **Пресет — під КОНКРЕТНИЙ формат, бо правила різні → прийоми можуть бути взаємовиключні.** Приклад (by-practice): **crossfeed L↔R** (підмішування протилежного каналу у фронт для стабілізації сцени) — застосовний під **EMMA**, але **НІКОЛИ під AYA** (там цінується природна ширина/сепарація). Повна стратегія пресетів (SQ/FULL/SQL/surround/source + per-ruleset) → `references/preset-strategy.md`.
- ⚠️ **Після перемикання пресета на змаганнях — перевірити що вхід DSP = ДЖЕРЕЛО прослуховування** (у цьому авто — **ОПТИКА S/PDIF**, не RCA; RCA лише для sweep-замірів). Перемикання пресета може скинути вхід на іншу карту (BT/USB) → нема/не той звук. Найприкріша втрата балів.

## EMMA 2026 — структура тест-диска (SQ Judgebook 2026; вижимка користувача, оригінал на сайті EMMA + NotebookLM)

> Велика зміна vs 2024: технічні треки ЧІТКО розділені за призначенням (позиції ≠ фокус ≠ сцена — окремі треки), музичний матеріал складніший.

**Трек 1 (Intro):** перше враження. Голос ВЕЛИКИЙ, точно центр → група труб/інструментів. Швидко, чисто, повно.
**Треки 2-3:** цілісність L/R каналів.
**Треки 2-6 — ПОЗИЦІЇ (тільки позиції: L/LC/C/RC/R).** На кожній позиції по черзі 5 інструментів: Electronic Bass (20-500 Гц) · Electronic Guitar (60-1200) · Banjo (160-5500) · Vibraphone (200-5000) · Triangle (2к-20к). Критерій: кожен інструмент = ОДНА маленька точка; «розмазаний»/чутний у кількох місцях = 0 балів.
**Треки 7-11 — ФОКУС (тільки розмір).** Ієрархія розмірів: бас НАЙБІЛЬШИЙ → гітара менша → банджо ще менше → вібрафон малий → трикутник НАЙМЕНШИЙ. Чітко, в межах сцени.
**Трек 12 — MOVING TRACK (сцена: ширина/висота/глибина/кімната).** Стабільні: барабани, флейта, шейкер, гонг, труба, орган, піаніно. Рухомі: muted guitar + cowbell. Критерії: відстань = найближчий інструмент за лобовим; висота = рівень очей/горизонту; кімнатна інформація через відлуння/ревер.
**Треки 13-16 — ТОНАЛЬНІСТЬ:**
- **13 «Ocean Drive»** — інтеграція НЧ. 0:18 ритм-секція (бас чіткий, контрольований, НЕ маскує СЧ); 0:40 жіночі голоси вплітаються, НЕ займаючи центрального фокуса.
- **14 «One Fine Day»** — реалізм вокалу/мікродинаміка. 0:29 вокал (чітко центр, трохи попереду інструментів); 0:39 бас зібраний, без «гудіння».
- **15 «Coming Back to You»** — глибокий чоловічий вокал чисто, без забарвлення; бек-вокалісти L і R від соліста, хор позаду.
- **16 «Should Have Done it Like This»** — динаміка. 2:03 хор = тест чіткості ВЧ + ширини; 2:23 труба (LC) і гітара (RC) чітко ВІДОКРЕМЛЕНІ.
**Спектральний баланс:** трек 16 на звичайній гучності, потім **+6 дБ** — спотворення/компресія на гучному = мінус бали.
**Трек 17 (Zero Bit):** тиша — без шипіння/гудіння/тріску/вентиляторів (двигун вимкнений).

**Порада посібника:** кожен інструмент у музичних треках — натурально й відокремлено, не впливаючи на інші.
