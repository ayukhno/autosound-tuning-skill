# VW Passat B8 sedan (limo) — карта кузова/салону

Профіль з реального проєкту (2026, LHD). На intake цього кузова — стартувати звідси, не передіагностувати з нуля; але **верифікувати на своїй машині** (інша інсталяція = інші install-залежні пункти).

## Аномалії КУЗОВА (очікувані незалежно від інсталяції)

- **Провал лівого мідбаса ~150 Гц** — геометрія/дифракція лівих дверей; НЕ minimum-phase → **EQ-буст заборонено**; працювати затримками/стиком із сабом.
- **Room-gain горб ~190 Гц** (седан).
- **Мідбас L/R антикорельований панч** (двері B8): w-L нуль ~150 / пік 90–127; w-R дзеркально протилежно. Не «лікувати» одну сторону під іншу наосліп.
- **Фазовий розрив L/R СЧ 230–320 Гц (~40°)** при паралельній посадці; мікро-затримки його не схлопують. АЛЕ: сумація показала, що міди там однаково сумуються когерентно (перевірено 2026-06-05) — не панікувати, перевірити сумацією у себе.
- Room gain седана: саб ЗЯ ~35 л у багажнику працює добре; +10..15 дБ нижче 50 Гц — норма.

## Аномалії ІНСТАЛЯЦІЇ (актуальні лише для схожої посадки)

- **СЧ на A-pillar → SBIR-нотчі R~645 / L~850 Гц** (мід стріляє в кут лобове↔торпедо; source-side, доведено рухом джерела). EQ-буст заборонено; поглинач лікує, але нежиттєздатний у їзді; **релокація в межах торпедо мертва** (розвідано: боки/kick/двері = гребінка; чистий лише центр торпедо, де стерео не живе). Шлях = DSP у межах обмеження + center-fill.
- A-pillar СЧ near-coplanar з ВЧ = найфорвардніша геометрія → стеля по глибині сцени (`staging-depth.md §4`).

## Що перемагало (кросовери/прийоми)

- **BE4 на стику СЧ↔ВЧ** (близькі, майже копланарні на стійці) — ключ конкурсної перемоги (AYA, молодша група): LPF СЧ 3k2BE4 + HPF ВЧ 5k0BE4 (акустичний стик ~4к через перекриття Bessel).
- **LR4 на мідбас↔СЧ** (~300 Гц електрично ≈ 250 акустично через спад мідбаса B8) — рознесені драйвери.
- Саб: LP ~60–63 LR4; субсонік 20BE1/BW2 (ЗЯ); стик SW+Ws судити сумацією.
- **Важкі мідбаси в дверях → зведення по 100% піку IR, не по носу** (`car-eq-patterns.md`).
- Свідома L/R асиметрія рівнів/затримок (LHD: ліві канали тихіше і затриманіші) → центрування.

> Джерело: проєкт автора скіла (Helix DSP Ultra S, 3-way фронт + саб + центр + тил). Повна історія — у проєктному `autosound_context.md` автора.
