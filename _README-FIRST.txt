Welcome to the manual car audio tuning pipeline!

You are reading this file as PLAIN TEXT. Every other file in this folder is
written in Markdown (.md) — the full protocol, the stateless-chat rule, and
speaker safety instructions all live in README.md, but as raw text it looks
cluttered with #, *, and | symbols. Read it with one of these instead:

* OPTION A (Recommended for PC): VS Code
  Open README.md in VS Code and press Cmd+Shift+V (macOS) or Ctrl+Shift+V
  (Windows) for a rendered preview.

* OPTION B (Free app, any OS): Obsidian (obsidian.md)
  Drag the whole "manual_step-by-step" folder into Obsidian as a vault — it
  renders tables, links, and the Mermaid diagram perfectly.

* OPTION C (Quick & offline): GitHub
  Open this branch's page in a browser — GitHub renders .md files natively.

* OPTION D (Browser-based): a "Markdown Viewer" browser extension
  Drag and drop any .md file straight into the browser window.

-------------------------------------------------------------------------------
Once you can read Markdown cleanly, start with README.md. It has the full
step-by-step protocol, the stateless-chat philosophy (why every step needs a
brand-new chat), and the speaker safety rules to read BEFORE your first REW
measurement.
