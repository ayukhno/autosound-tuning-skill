# Security Policy

If you discover a security vulnerability in this repository, thank you for responsibly
reporting it so we can address it quickly.

Preferred reporting channels

1. GitHub Security Advisories: https://github.com/ayukhno/autosound-tuning-skill/security/advisories
   - This lets maintainers coordinate a fix and a coordinated disclosure.

2. Email (alternate): 60683988+ayukhno@users.noreply.github.com
   - Include: affected file(s), reproduction steps, severity, proof-of-concept (if safe), and any suggested remediation.

How we handle reports

- We acknowledge receipt within 3 business days.
- We'll triage and communicate a timeline for remediation.
- We will coordinate disclosure timing and provide credit in the repository unless you request anonymity.

Do not publish vulnerability details publicly until a fix or coordinated disclosure is agreed.

PGP / encrypted reports

- If you prefer to send an encrypted report, please request the public key via the GitHub Security Advisory or the email above and we will provide instructions.
